Competence Project REST API

This document defines the REST API for the Competence Project, which allows the management of evaluations for kindergarten students. The API provides access to add, retrieve, and manage data about students, evaluations, results, and more.

Base URL

Base URL: to be defined

Endpoints Overview

  • /eleves/ - Manage students (Eleves)
  • /resultats/ - Manage evaluation results (Resultats)
  • /catalogues/ - Manage evaluation catalogues (Catalogues)
  • /items/ - Manage evaluation items (Items)
  • /niveaux/ - Manage class levels (Niveaux)
  • /etapes/ - Manage evaluation stages (Etapes)
  • /annees/ - Manage academic years (Annees)

/eleves/ - Manage Students (Eleves)

API endpoint to create, retrieve, update, and delete student (Eleve) records.

GET /eleves/
POST /eleves/

/resultats/ - Manage Evaluation Results (Resultats)

API endpoint to manage the evaluation results of students.

GET /resultats/
POST /resultats/

/catalogues/ - Manage Evaluation Catalogues (Catalogues)

API endpoint to manage evaluation catalogues, which define the structure of evaluations.

GET /catalogues/
POST /catalogues/

/items/ - Manage Evaluation Items (Items)

API endpoint to manage individual evaluation items within a catalogue.

GET /items/
POST /items/

/niveaux/ - Manage Class Levels (Niveaux)

API endpoint to manage class levels, such as CP, E1, and M1.

GET /niveaux/
POST /niveaux/

/etapes/ - Manage Evaluation Stages (Etapes)

API endpoint to manage the different stages of evaluations (e.g., Debut, Inter, Final).

GET /etapes/
POST /etapes/

/annees/ - Manage Academic Years (Annees)

API endpoint to manage academic years (e.g., 2024-2025).

GET /annees/
POST /annees/
